\problem\problemscore{10}
Положительные числа $x$, $y$ и $z$ таковы, что $x + y + z = 1$.
Докажите неравенство
\[
    \frac{x y}{\sqrt{z + x y}}
    +
    \frac{y z}{\sqrt{x + y z}}
    +
    \frac{z x}{\sqrt{y + z x}}
\leq
    \frac{1}{2}
.\]
\solution
Заметим, что
\[
    (z + x) (z + y)
=
    z^2 + x z + y z + x y
=
    z (x + y + z) + x y = z + x y
.\]
Значит, для первого слагаемого мы имеем:
\[
    \frac{x y}{\sqrt{z + x y}}
=
    \frac{x y}{\sqrt{(z + x) (z + y)}}
\leq
    \frac{1}{2} \left(
        \frac{xy}{z + x} + \frac{xy}{z + y}
    \right)
.\]
Суммируя все такие неравенства, мы получаем требуемое.
\endproblem
